Программа является интерпретатором языка P#.
Возможности программы:

- Вы можете использовать несколько программ одновременно, добавляя страницы при нажатии на "+";

- Также доступны три стиля во вкладке "Style": светлый, неоновый и кошачий;

- Наберите код в левом окне и нажмите кнопку "Comp". Если код не содержит ошибок, то вывод программы отобразится в правом верхнем окне. В ином случае список ошибок отобразится в правом нижнем окне;

- Читайте из файла и записывайте в файл код, используя кнопку "File";

- Для вызова подсказки воспользуйтесь "Help";

- Вы можете отключить звук через кнопку "Sound".

Краткое описание синтаксиса языка P#.

- В языке присутствует два типа данных - полиномы и матрицы от полиномов;

- Введение новой переменной типа Polynomial задаётся в виде: "pol [name_var];" или "pol [name_var] := [calculating_expression];". В первом варианте полином по умолчанию имеет нулевое значение;

- Полиномы записываются в виде "a_1*x^1+a_2*x^2+...+a_n*x_n";

- Матрица задаётся выражением: "matrix [name_var] := ([height_matrix],[width_matrix]);". Введённая матрица имеет нулевые коэффициенты;

- Присваивание значения матрице и полиному имеет вид: "[name_var] := [calculating_expression];", а коэффициенту матрицы - "[name_matrix][[height_matrix]][[width_matrix]] := [calculating_expression];";

- Чтобы вывести полином используйте оператор "write([calculating_expression]);";

- Оператор if ([bool_expression]) при верном логическом выражении выполняет операторы, заключённые в фигурные скобки, находящиеся после оператора;

- Оператор "while ([bool_expression])" при верном логическом выражении выполняет цикл, заключённый в фигурные скобки, находящиеся после оператора;

- Вычислимые выражения содержат операторы +, -, *, /, %, Diff(дифференцирование), EvalP(подстановка полинома в полином), EvalM(подстановка матрицы в полином), ChPol(нахождение характеристического полинома матрицы), Rand(случайный полином) для вычисления полиномов и +, -, *, coef(коэффициент матрицы по индексу), det(определитель матрицы) для матриц;

- Логические выражения содержат операторы >, < >=, <=, ==, !=, !, |, &. сравнение полиномов должно быть обрамлено в скобки;

- После инициализации, присваивания, вывода и "goto" необходима точка с запятой;

- Операторы "goto([number_command])"(переход на команду с некоторым номером) и "to"(переход на одну команду вперёд) являются вспомогательными, хотя и допускаются в коде.
